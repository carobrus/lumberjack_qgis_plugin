NAME

r.out.haralickFeatures - Compute asm, contrast, var, idm and entr for each band of a TIFF file.

KEYWORDS

raster, haralick, texture

SYNOPSIS

r.out.haralickFeatures
r.out.haralickFeatures --help
r.out.haralickFeatures [-r] input=name size=value distance=value categories=value output=name [--overwrite] [--help] [--verbose] [--quiet] [--ui]

Flags:

-r
Rescales instead of recoding the image
--overwrite
Allow output files to overwrite existing files
--help
Print usage summary
--verbose
Verbose module output
--quiet
Quiet module output
--ui
Force launching GUI dialog

Parameters:

input=name [required]
Name of input raster map
size=value [required]
The size of moving window (odd and >= 3)
Default: 3
distance=value [required]
The distance between two samples (>= 1)
The distance must be smaller than the size of the moving window
Default: 1
categories=value [required]
Number of categories to rescale/recode the image
Must be smaller than 256
Default: 255
output=name [required]
Name of output file (full path)
